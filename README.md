**LiftMosque** is a premium community carpooling application designed to facilitate travel to mosques for prayer. It connects drivers with available seats to passengers going to the same mosque, fostering community solidarity and eco-friendly transportation.

## ğŸš€ Features

### Mobile Application (Flutter)
- **Real-time Trip Discovery**: Browse and search for available rides to your local mosque.
- **Offer a Ride**: Drivers can easily publish their trips with departure points, pickup stops, and available seats.
- **Dynamic Mosque List**: Real-time synchronization with a centralized mosque database.
- **Notifications**: Stay updated on trip requests and confirmations.
- **Premium UI**: Clean, modern design with a focused "Green & Blue" theme.

### Admin Dashboard (Vite + React)
- **Real-time Analytics**: Monitor user growth, active trips, and mosque counts.
- **Mosque Management**: CRUD operations to manage the list of supported mosques globally.
- **Trip Monitoring**: Review and manage active trips across the platform.
- **Secure Authentication**: Dedicated admin access via Firebase Auth.
- **Modern Stack**: Built with Tailwind CSS v4 and official Vite integration.
